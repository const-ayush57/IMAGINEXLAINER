# Design Document: Visuals & UI/UX

## 1. Vibe & Aesthetic
- **Theme:** Modern, sleek, "AI-native". Dark mode default to reduce eye strain and look professional.
- **Keywords:** Minimalist, fast, frictionless, magical.
- **Typography:** Inter or Roboto for clean, highly readable interfaces.

## 2. Color Palette
- **Background:** Very dark gray/black (e.g., `#121212` or `#0F172A`).
- **Primary Accent:** Electric Purple (`#8B5CF6`) or Cyan (`#06B6D4`) for the main "Generate" buttons and loading spinners.
- **Text:** Off-white (`#F8FAFC`) for primary headers, soft gray (`#94A3B8`) for secondary text.

## 3. Key Interface Components
- **Hero Section:** Large, bold headline ("Presentations that Speak for Themselves"). Centered, prominent text input field for the prompt.
- **Loading State:** Crucial for user retention. Needs a sleek progress bar or step-by-step checklist that lights up as the backend processes (Writing -> Speaking -> Compiling).
- **Result View:** A split view showing the 2-slide preview download on one side, and a clearly marked premium tier (with Stripe integration) on the other.

## 4. Responsiveness
- Must be fully functional on mobile devices, though the primary use case (downloading PPTX) implies a desktop-first layout for the final result screen.
